Course Description

This course introduces toWeb technologies-HTML,CSS,JavaScript,CSS Framework; JavaScript Frameworks.Studends will learn how to implement modern web using HTML and CSS Framework. the coursea lso introduses students to full stack web application programming using Javascript.

Course Objectives

  1. Describe web technologies (HTML, CSS and JavaScript) CSS Frameworks, and JavaScript Frameworks
  2. implement a simple web page with HTML and CSS
  3. Build a simple but functional web application using JavaScript
